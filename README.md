# Quick Start  

1) Clone the repository  
2) Enter MapD server details in `./src/index.js`  
        `connector.protocol(protocol)`  
        `.host(hostname)`  
        `.port(port)`  
        `.dbName(dbname)`  
        `.user(username)`  
        `.password(pwd)`  
3) Install yarn:  
        `brew install yarn`  
4) Load dependencies:  
        `yarn install`  
5) Start Applciation:  
        `yarn run start`  
        
The application is running @ http://localhost:8080/
